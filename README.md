# 🏋️ Sport Fit – Personalized Fitness System

Sport Fit is a web-based fitness application designed to provide **personalized workout and nutrition plans** based on a user’s sport and fitness goals. The system uses a **rule-based AI engine** along with structured datasets to deliver sport-specific recommendations.

---

## 📌 Abstract

Sport Fit is a modern web application that offers customized fitness solutions for athletes and general users. It generates workout and diet plans based on user preferences and sports categories. The system also includes an AI-powered chat assistant for instant fitness guidance, making it interactive, scalable, and user-friendly.

---

## 🚨 Problem Statement

- No centralized platform for sport-specific fitness guidance  
- Difficulty managing large fitness datasets  
- Existing apps provide **generic recommendations**  
- Poor scalability and lack of modular architecture  

---

## 💡 Proposed Solution

- Modular web-based system  
- AI-based sport-wise workout recommendation  
- Integrated AI fitness chatbot  
- Scalable and maintainable architecture  
- Separation of frontend, backend, and database layers  

---

## ✨ Features

- 🏃 Personalized workout plans

---

## 🧩 Modules

### 1. Fitness Recommendation Module
- Generates workout & diet plans  
- Uses rule-based logic  

### 2. Sports Information Module
- Provides sport-specific training guidance  

### 3. AI Chat Assistant Module
- Answers fitness-related queries  
- Helps with injury prevention and training  

### 4. Database Module
- Stores user data, workouts, and diet plans  

### 5. Authentication Module
- Handles login and security using JWT  

### 6. Dashboard Module
- Displays personalized data and progress
